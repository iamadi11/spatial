# Release 0.9.0 — 2026-04-03

## What's New

### [021] Report summary formatter

Added `formatReport(result: PerformanceResult): string` in `src/report-summary.ts`.

Renders a complete `PerformanceResult` as a multi-line text block:
- `Status: PASS|FAIL|UNKNOWN`
- `Metrics: renderCount=N layoutShifts=N fpsDrop=N memoryUsage=N`
- `Reason: ...` (only when present)
- `Issues: none` or issue list via `formatIssues()`

- Pure function, builds on `formatIssues()` from item 012
- No HTML, no ANSI codes, no DOM — deterministic text only
