# Release 0.5.0 — 2026-04-03

## What's New

### [015] Style complexity detection rule

Added `createStyleComplexityRule(): Rule` in `src/rules/style-complexity.ts`.

Flags any component node whose `styles` object contains known GPU-compositing expensive CSS properties. Addresses the "dynamic CSS" risk case in SOT Section 4.2.3.

**Flagged properties**: `boxShadow`, `filter`, `backdropFilter`, `transform`, `clipPath`

- Pure function — key-match only against a static set, no value evaluation, no DOM
- Severity: `warning`
- Message lists all expensive properties found
- Deterministic: metric values have no effect on the result
