# Release 0.3.0 — 2026-04-03

## What's New

### [013] Child count detection rule

Added `createChildCountRule(threshold?: number): Rule` in `src/rules/child-count.ts`.

Flags any component node whose number of direct children exceeds the configured threshold (default: 20). Addresses the "Large component trees" and "Long lists without virtualization" edge cases called out in SOT Section 4.2.2.

- Pure function — reads `node.children.length` only, no DOM, no randomness
- Severity: `warning`
- Deterministic: metrics values have no effect on the result
