# Release 0.4.0 — 2026-04-03

## What's New

### [014] Prop count detection rule

Added `createPropCountRule(threshold?: number): Rule` in `src/rules/prop-count.ts`.

Flags any component node whose `props` object has more keys than the configured threshold (default: 15). Addresses the "Large component trees" edge case in SOT Section 4.2.2 — excessive props are a direct re-render risk proxy.

- Pure function — reads `Object.keys(node.props ?? {}).length` only, no DOM, no randomness
- Severity: `warning`
- Deterministic: metric values have no effect on the result
