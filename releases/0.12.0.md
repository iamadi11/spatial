# Release 0.12.0 — 2026-04-03

## What's New

### [026] Duplicate component type detection rule

Added `createDuplicateComponentTypeRule(threshold=30)` in `src/rules/duplicate-component-type.ts`.

Tree-level rule (uses `check(root)` pattern, same as nesting-depth and total-node-count):
- Single O(n) DFS pass counts occurrences of each component type
- Only Pascal-cased types are counted (HTML tags like `div`, `span` are ignored)
- Flags the most-repeated type that exceeds the threshold
- Returns one `PerformanceIssue` with the type name, count, and threshold in the message
- Severity: `warning`
