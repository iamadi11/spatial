# Release 0.9.1 — 2026-04-03

## What's New

### [022] React fiber adapter

Added `extractTree(fiber: unknown): ComponentNode` in `src/adapters/react.ts`.

Bridges the live React fiber tree to the engine's `ComponentNode` input contract:
- Reads `fiber.type`, `fiber.memoizedProps`, `fiber.child`, `fiber.sibling` recursively
- Component name resolved from function `.name` or string tag; falls back to `'unknown'`
- Function props omitted from extracted props (not serialisable)
- Unique id assigned to every node via index path (e.g. `"0"`, `"0.0"`, `"0.0.1"`)
- Dev-only guard: returns `{ id: 'root', type: 'unknown' }` in production
- Read-only: never mutates the fiber tree
