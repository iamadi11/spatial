# Release 0.9.2 — 2026-04-03

## What's New

### [023] Metrics adapter

Added `collectMetrics(renderCount: number): PerformanceMetrics` in `src/adapters/metrics.ts`.

Collects a live snapshot of all four engine metrics from browser APIs:
- `layoutShifts`: accumulated via `PerformanceObserver` (`type: 'layout-shift'`)
- `fpsDrop`: rolling `requestAnimationFrame` loop counting frames > 2× 16.7ms target
- `memoryUsage`: `performance.memory.usedJSHeapSize / 1_048_576` (MB); 0 if unavailable
- `renderCount`: injected by the caller (React Profiler owns render instrumentation)

Guards:
- Returns `{ renderCount:0, layoutShifts:0, fpsDrop:0, memoryUsage:0 }` in production
- Safe no-op when `PerformanceObserver` or `requestAnimationFrame` is unavailable

Also exports `resetMetrics()` to zero module-level accumulators between test runs.
