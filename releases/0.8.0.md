# Release 0.8.0 — 2026-04-03

## What's New

### [018] Inline style count detection rule

Added `createInlineStyleCountRule(threshold?: number): Rule` in `src/rules/inline-style-count.ts`.

Flags any component node whose `styles` object has more keys than the configured threshold (default: 15). Targets the per-render parsing overhead of excessive inline style declarations.

Distinct from `style-complexity` (which flags specific expensive property names) and `prop-count` (which flags the props object).

- Pure function — reads `Object.keys(node.styles ?? {}).length` only
- Severity: `warning`
- Deterministic: metric values have no effect
