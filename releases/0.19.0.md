# Release 0.19.0

**Date:** 2026-04-09

## Added

- **033 — Multi-type sibling fan-out rule** (`src/rules/multi-type-sibling-fanout.ts`): warns when a node has at least 10 direct children and at least 6 distinct child `type` values among those children. Complements `child-count` and `unvirtualized-list`. Dashboard adapter registers the rule and exposes catalog metadata plus optional `multiTypeSiblingFanoutMinChildren` / `multiTypeSiblingFanoutMinDistinctTypes` in `RuleOptions`.
