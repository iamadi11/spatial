# Release 0.8.2 — 2026-04-03

## What's New

### [020] Rule catalog

Added `createRuleCatalog(): RuleMetadata[]` in `src/rule-catalog.ts`.

Returns structured metadata for all 10 built-in rules: name, description, severity, and defaultThreshold (absent for `style-complexity` which uses a fixed property set).

- Pure function — same output every call, no side effects, O(1)
- Enables dashboard and tooling to describe the engine's capabilities without reading source
