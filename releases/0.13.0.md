# Release 0.13.0 — 2026-04-03

## What's New

### [027] Large data prop detection rule

Added `createLargeDataPropRule(thresholdBytes=10_000)` in `src/rules/large-data-prop.ts`.

Per-node rule (uses standard `detect()` interface):
- Estimates serialised byte size of non-function props via `JSON.stringify`
- Function-valued props are excluded from the size calculation
- Circular references are caught safely — that prop counts as 0 bytes
- Flags nodes where total payload exceeds `thresholdBytes`
- Severity: `warning`, message includes approximate byte count and threshold
