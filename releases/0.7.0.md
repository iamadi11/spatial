# Release 0.7.0 — 2026-04-03

## What's New

### [017] Total node count detection rule

Added two exports in `src/rules/total-node-count.ts`:

- **`countNodes(root: ComponentNode): number`** — pure O(n) DFS utility that counts all nodes in the tree
- **`createTotalNodeCountRule(threshold?: number): TotalNodeCountRule`** — tree-level checker returning a `PerformanceIssue` when total nodes exceed the threshold, or `null` otherwise

Addresses "Very large component trees" in SOT Section 4.2.2. Complements the existing `child-count` (width) and `nesting-depth` (depth) rules with an overall tree-size dimension.

- Default threshold: 200
- Severity: `warning`
- Deterministic: same tree always produces same result
