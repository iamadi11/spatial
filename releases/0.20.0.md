# Release 0.20.0

**Date:** 2026-04-09

## Added

- **034 — classname-token-sprawl** (`src/rules/classname-token-sprawl.ts`): warns when `props.className` is a string and either the trimmed length exceeds 400 or the whitespace-separated token count exceeds 30 (defaults). `createClassnameTokenSprawlRule({ maxTokens, maxLength })`. Dashboard `runAnalysis` registers the rule; optional `RuleOptions` fields `classnameTokenSprawlMaxTokens` and `classnameTokenSprawlMaxLength`.
